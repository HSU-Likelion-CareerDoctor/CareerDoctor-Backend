### 📢 커밋 <br>
#### 1️⃣ 커밋 룰
- 최소 작업 단위로 가능한 작게 쪼갠다.
- 1개의 커밋에는 1개의 행위만 들어 있게 한다.
- 아래의 작성 규칙을 따른다. <br>
  - 커밋 타이틀 첫 문자는 대문자로 작성한다.
  - 제목은 명령문으로 사용하며 과거형을 사용하지 않는다.
  - 2️⃣의 커밋 타입을 작성한다.
  - 예시 : Feat. 회원가입 API 작성
